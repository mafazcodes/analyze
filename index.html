Welcome to the Data Processing Project

This project demonstrates a robust data processing pipeline built with Python, Pandas, and GitHub Actions.

Explore the repository to understand the implementation details, including the `execute.py` script, `data.csv`, and the CI/CD workflow (`.github/workflows/ci.yml`). The processed output, `result.json`, is automatically generated and deployed.

View Source CodeCheck CI/CD Status

Data Ingestion

Starts with `data.xlsx`, converted to `data.csv` for efficient processing.

Core Logic

The `execute.py` script, written in Python 3.11+ with Pandas 2.3, handles data transformation and analysis.

Automation

GitHub Actions (`ci.yml`) automates linting, execution, and deployment of `result.json` to GitHub Pages.
